# إبراز إجراءات مركز المشاريع الأساسية

التاريخ: 2026-03-09

## الهدف
تحسين وضوح تجربة المستخدم في مركز المشاريع عبر إبراز إجراءين أساسيين بشكل مباشر:
- إنشاء مشروع جديد
- تعديل اسم المشروع

## الدافع
كانت الإجراءات موجودة لكن متفرقة أو داخل نافذة "إدارة المشاريع"، مما يصعّب اكتشافها للمستخدم غير الخبير.

## النطاق
- تعديل واجهة `app/strategy-workspace.tsx` فقط.
- بدون تغيير في منطق البيانات أو الصلاحيات الأساسية.

## ما تم تنفيذه
1. إضافة إجراء مركزي جديد في المنطق:
   - `requestRenameProjectById(projectId)`
   - يدعم تعديل اسم أي مشروع نشط مباشرة من مركز المشاريع.
   - يحافظ على نفس صلاحيات التعديل الحالية.

2. النسخة التجريبية من مركز المشاريع:
   - إضافة زرين واضحين في الشريط العلوي:
     - "إنشاء مشروع جديد"
     - "تعديل اسم المشروع"
   - إضافة زر "تعديل الاسم" داخل بطاقة كل مشروع نشط.

3. النسخة الكلاسيكية من مركز المشاريع:
   - إضافة شريط إجراءات واضح أعلى الشبكة:
     - "إنشاء مشروع جديد"
     - "تعديل اسم المشروع"
   - إضافة زر "تعديل الاسم" داخل بطاقة كل مشروع نشط.

4. تحديث التخطيط:
   - توسيع شبكة أزرار البطاقات لتستوعب زر التعديل الجديد.
   - الحفاظ على توافق العرض مع الجوال.

## التحقق
- `npm run lint`: ناجح.
- `npm run build`: ناجح.

## الأثر على المستخدم
- المستخدم يرى الإجراءات الأساسية مباشرة عند فتح مركز المشاريع.
- تقليل الحاجة لفتح نوافذ إضافية لتنفيذ عمليات متكررة.
